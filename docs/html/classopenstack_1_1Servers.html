std::auto_ptr<xml::Servers> openstack::Servers::_list ( bool  details = false) const [inline, protected]

Downloads and parses something like this:

<?xml version="1.0" encoding="UTF-8" standalone="yes"?> <servers xmlns="http://docs.rackspacecloud.com/servers/api/v1.0"> <server id="1234" name="admin"> <server id="563" name="stats"> <server id="456546" name="web2"> <server id="2345345" name="web1"> <server id="234345" name="db1"> <server id="34788" name="db2"> </servers>"

To use the return value: const xml::Servers::ServerSequence oc.servers().list().server(); for(xml::Servers::ServerConstIterator server=list.begin();server!=list.end();++server) std::cout << "ID: " << server->id() << " - " << "Name: " << server->name() << std::endl;

Parameters:
detailsif true, will get all the server details. If false, will get just the IDs and names
Returns:
A smart pointer to an xml::Servers instance.
